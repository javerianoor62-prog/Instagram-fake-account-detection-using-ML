Project Title
Instagram fake account detection using ML
Project Brief
This project develops an intelligent system using machine learning to detect fake Instagram accounts. It analyzes features like follower-following ratio, post frequency, engagement rate, and bio content to classify genuine or fake users. A web interface allows testing account data and visualizing detection results.
Technologies
Backend: Python (Flask)
Frontend: HTML, CSS, JavaScript
MLA Tools: Loggistic regression,Random Forest,XGBoost
Dataset: Istagram public profiles(Kaggle dataset or custom-scraped data)
Type
AI + MLA + Web
